# Remove-KshSite

## SYNOPSIS
Removes a site.

## SYNTAX

```
Remove-KshSite [-Identity] <Site> [<CommonParameters>]
```

## DESCRIPTION
The Remove-KshSite cmdlet removes a site from the current site collection.

## EXAMPLES

### Example 1
```powershell
PS C:\> Remove-KshSite (Get-KshSite -SiteId 'd298e576-6985-4119-9796-050b9f371872')
```

Removes a site.
