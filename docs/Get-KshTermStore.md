# Get-KshTermStore

## SYNOPSIS
Retrieves the term store.

## SYNTAX

```
Get-KshTermStore [<CommonParameters>]
```

## DESCRIPTION
The Get-KshTermStore cmdlet retrieves the term store.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-KshTermStore
```

Retrieves the term store.
