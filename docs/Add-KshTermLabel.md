# Add-KshTermLabel

## SYNOPSIS
Creates a new term label.

## SYNTAX

```
Add-KshTermLabel [-Term] <Term> -Name <String> -Lcid <UInt32> -IsDefault <Boolean> [<CommonParameters>]
```

## DESCRIPTION
Add-KshTermLabel cmdlet adds a new label to the term.

## EXAMPLES

### Example 1
```powershell
PS C:\> Add-KshTermLabel -Term (Get-KshTerm -TermSet (Get-KshTermSet -TermGroup (Get-KshTermGroup -TermGroupName 'Company') -TermSetName 'Department') -TermName 'Human Resources') -Name 'HR' -Lcid 1033 -IsDefault $false
```

Creates a new term label.
