# Get-KshCurrentSite

## SYNOPSIS
Retrieves the current site.

## SYNTAX

```
Get-KshCurrentSite [<CommonParameters>]
```

## DESCRIPTION
The Get-KshCurrentSite cmdlet retrieves the current site.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-KshCurrentSite
```

Retrieves a current site.
