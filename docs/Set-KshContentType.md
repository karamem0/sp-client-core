# Set-KshContentType

## SYNOPSIS
Updates a content type.

## SYNTAX

```
Set-KshContentType [-Identity] <ContentType> [-Description <String>] [-DisplayFormUrl <String>]
 [-EditFormUrl <String>] [-Group <String>] [-JSLink <String>] [-Hidden <Boolean>] [-Name <String>]
 [-NewFormUrl <String>] [-ReadOnly <Boolean>] [-Sealed <Boolean>] [-PassThru] [<CommonParameters>]
```

## DESCRIPTION
The Set-KshContentType cmdlet updates properties of the content type.

## EXAMPLES

### Example 1
```powershell
PS C:\> Set-KshContentType -Identity (Get-KshContentType -ContentTypeId '0x0100EFB1758564C77D448177233D1199B912') -ReadOnly $true
```

Updates property values of the content type.
